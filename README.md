# Atom One Dark theme for Terminal
A theme for Terminal and iTerm that mimics the native [One Dark Theme](https://atom.io/themes/one-dark-syntax) made by the Atom team. Slight color changes were made between One Dark versions 0.4.0 and 0.5.0; both of these are supported.
